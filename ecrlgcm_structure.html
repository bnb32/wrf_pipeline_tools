Code Structure

This page briefly describes the ecrlgcm main components.

Preprocessing

ecrlgcm/preprocessing/__init__.py contains functions to remap continents and compute initial conditions.

Postprocessing

ecrlgcm/postprocessing/__init__.py contains functions used in the interactive dashboard and creating gifs.
To run the interactive dashboard use python ecrlgcm/postprocessing/app.py.
To generate gifs use ecrlgcm/postprocessing/get_animation.py.
To generate an html file with an interactive globe use ecrlgcm/postprocessing/get_interactive_globe.py.

Data

ecrlgcm/data/ contains historical data on CO2 and orbital parameters.

Simulation

scripts/run_cesm.py and scripts/run_isca.py are used to run simulations.
