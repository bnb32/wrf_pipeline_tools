Examples

Here are few examples on how to run paleoclimate simulations with Isca or CESM using the ecrlgcm package and to visualize the output.

Run Simulations

Running a CESM simulation for 300Ma BP:

python scripts/run_cesm.py -year 300.0 -run_all

Running an Isca simulation for 300Ma BP:

python scripts/run_isca.py -year 300.0

Visualization

Run interactive dashboard:

python postprocessing/app.py

Note

These examples provide practical ways to illustrate the usage of the software, and do not necessarily represent the best scientific methods for any particular type of study.
